Here is what I have been able to scrape together. Most of my experience as a full-stack developer has been with Ruby on Rails with React and ERB for front end. Due to this I was unable to get as much done as I would have liked using Node.js, Express and Vue.js. This was really fun though!

To run this you need to:

1. Run npm install twice, once in the webapp and another time in the webservice
2. Once this is done, run npm start dev in the webapp and npm start in the webservice.
3. Go to http://localhost:8080 for the front end and http://localhost:8081 for the back end.
4. Enjoy just the first page and the console log showing that the backend was hit!
